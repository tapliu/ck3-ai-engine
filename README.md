# 浪子江湖 · 侠行四海

AI 驱动的武侠江湖模拟游戏。35 位角色出自小说《浪子江湖》，每人有机敏、武力、魅力、智谋四项属性及年龄、描述。

## 快速开始

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

浏览器打开 `http://localhost:8000`

> 头像和全身像 PNG 图片放在 `data/Images/portrait/` 和 `data/Images/full-body/` 目录下，自动挂载到 `/api/images/`。

## 游戏玩法

1. **选人** — 开局从 3 个随机角色中选择你的侠客（带全身像预览）
2. **Tick** — 每点击一次推进一回合，NPC 会自主决策（战争、刺杀等）
3. **任务** — 每回合出现 1–3 个江湖任务，分四类：
   - **非人物类** — 成功主属性：机敏
   - **人物类** — 成功主属性：魅力
   - **合作任务** — 同城 50%/同区域 25% 概率将普通任务转为合作，成功 +20~40 好感，失败 -10~20
   - **极难任务** (30% 概率) — 高风险高回报，失败可能死亡，成功属性 +3~5
   - 三种执行模式：
     - **普通** — 属性 × 0.5 计算成功率
     - **用谋略** — 普通 + 智谋 × 0.1 加成
     - **全力施展** — 属性 × 0.75，成功率最高
4. **事件** — 好感度 ≥ 60 且满足条件角色 ≥ 3 人时触发拜师、结婚、结拜等人生大事
5. **奇遇** — 玩家 30%/NPC 20% 概率获得宝物，属性随之成长；宝物按类别（兵器/秘籍/宝物/灵药）独占，同类新宝物替换旧宝物
6. **侠义值** — 完成任务成功增加 2~6 侠义值，影响武道会参赛资格
7. **不进则退**（每 10 回合）— 选择一项属性降低 10%
8. **天下第一武道会**（每 30 回合）— 侠义值 Top 16 参赛，四人小组双败淘汰 → 8 强淘汰赛 → 冠军获「天下第一人」称号，武力 +10，众人好感 -10
9. **战斗死亡** — 同城市关系 < -5 爆发战斗，胜负由武力决定，武力差越大死亡概率越高（5%~50%），主角死亡则游戏结束
10. **江湖移动**（每 2 回合）— 角色按城市路线图移动。主角待选时出现路线弹窗，手动选择目的地；NPC 随机选择相邻城市自动移动。跨区域时自动更新所属区域
11. **江山舆图** — 弹窗展示地图图片及 18 城详情（区域/人口/关隘/环线连接/帝都/要冲标签），中原 6 城左右分列，其余四区 2×2 网格排列
12. **称号系统** — 13 种称号（初出茅庐、侠义之士、一代大侠、武林高手、百战英雄等），达成条件自动获得
13. **结算画面** — 游戏结束时展示：称号、属性变化、好友宿敌、宝物、江湖历程时间线
14. **托管模式** — 点击「托管」自动执行每回合，自动处理移动/任务/事件/衰减，回合完成后停止

## 城市系统

18 座城市分布于五大区域，角色出生时均衡分配至所属区域内的城市：

| 区域 | 城市 |
|------|------|
| 中原 | 洛阳、开封、长安、燕京、应天府**(帝都)**、襄阳 |
| 江南 | 苏州**(中心→襄阳)**、杭州、扬州 |
| 塞北 | 凉州**(中心→燕京)**、朔方、云中 |
| 西域 | 敦煌**(中心→长安)**、龟兹、于阗 |
| 东海 | 登州**(中心→应天府)**、泉州、宁波 |

中原四座关隘城市（长安、燕京、应天府、襄阳）连成环形：长安—燕京—应天府—襄阳—长安。东南关隘（应天府、襄阳）连接开封，西北关隘（长安、燕京）连接洛阳，开封与洛阳连通。

江南、塞北、西域、东海三城内部互相连通（全连接），方便区域内移动。

区域中心通过关隘连接中原（登州→应天府，苏州→襄阳，敦煌→长安，凉州→燕京）。

外部区域之间形成环线连接（东-南，南-西，西-北，北-东）：
- 泉州↔扬州（东海↔江南）
- 杭州↔于阗（江南↔西域）
- 龟兹↔朔方（西域↔塞北）
- 宁波↔云中（东海↔塞北）

战斗与和睦事件以城市为单位触发，同一城市的角色更易产生交互。

合作任务由普通任务按概率转化：**同城市 50%** / **同区域 25%** 转为合作，目标加权同城市 5 倍 > 同区域 2 倍。

## 移动系统

角色每 2 回合可按城市连接路线移动：

- **玩家** — 回合推进后若有 `pending_move`，弹出路线选择弹窗，手动点击目的地
- **NPC** — 自动随机选择一条相邻路线移动，跨区域时自动更新 `region`
- 托管模式下自动选第一条路线移动
- 城市连接定义在 `app/models/region.py` 的 `CITY_CONNECTIONS` 字典中，每座城市一条对外通道，无枢纽复用

## 属性系统

- **机敏(l)** — 非人物类任务成功率
- **武力(w)** — 战斗力，武道会决胜关键；≥150 获「武道宗师」
- **魅力(i)** — 人物类任务成功率
- **智谋(p)** — 谋略任务加成（系数 × 0.1）
- **侠义值** — 通过任务积累，影响武道会资格（需 ≥50 获「侠义之士」，≥100 获「一代大侠」）
- 属性按 `总值(基础值+加成值)` 分开展示（如 107 = 95 + 12）

## 好感度系统

- 初始好感随机范围 -20~20
- 合作任务成功 +10~20，失败 -5~10
- 拜师/结婚/结拜需好感 ≥ 60
- 同城市关系 < -5 可能爆发战斗，> 5 可能和睦互助（城市级交互，非区域级）

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/world` | 获取世界状态（回合、角色、玩家、好感度、结算数据） |
| GET | `/api/start/candidates` | 获取初始选人候选（3人） |
| POST | `/api/start/select` | 选择角色 `{"name":"黄宇翔"}` |
| POST | `/tick` | 推进一回合 |
| GET | `/api/events/pending` | 获取待处理大事（拜师/结婚/结拜） |
| POST | `/api/events/decide` | 处理事件 |
| GET | `/api/events` | 获取江湖日志 |
| GET | `/api/tasks/pending` | 获取本回合任务列表 |
| POST | `/api/tasks/execute` | 执行任务 `{"task_id":0,"mode":"normal\|scheme\|focus"}` |
| GET | `/api/decay/pending` | 获取不进则退待选属性 |
| POST | `/api/decay/execute` | 选择降低属性 `{"field":"base_w"}` |
| GET | `/api/tournament` | 获取武道会对阵图及结果 |
| GET | `/api/move/pending` | 获取待选移动路线（主角可移动时） |
| POST | `/api/move` | 执行移动 `{"dest":"洛阳"}` |
| GET | `/api/data/map.png` | 江山舆图地图图片 |
