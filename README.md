# 浪子江湖 · 侠行四海

AI 驱动的武侠江湖模拟&策略游戏。35 位角色出自小说《浪子江湖》，具备战斗、经济、城池经营与扩张等系统。

## 快速开始

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

浏览器打开 `http://localhost:8000`

> 头像和全身像 PNG 图片放在 `data/Images/portrait/` 和 `data/Images/full-body/` 目录下，自动挂载到 `/api/images/`。

## 游戏玩法

1. **选人** — 开局从 3 个随机角色中选择你的侠客（带全身像预览）
2. **Tick** — 每点击一次推进一回合，NPC 自主决策（战争/刺杀/结盟等）
3. **任务** — 每回合 1–3 个江湖任务，分六类：
   - **非人物类** — 成功主属性：机敏
   - **人物类** — 成功主属性：魅力
   - **合作任务** — 同城 50%/同区域 25% 概率将普通任务转为合作
   - **极难任务** (30% 概率) — 高风险高回报，失败可能死亡
   - **招募士兵**（概率触发）— 魅力影响成功率，每次 +500 兵力
   - **攻城略地**（兵力 > 2000 时概率出现）— 从所在地攻打相邻城池
   - 三种执行模式：普通 / 用谋略（智谋加成）/ 全力施展
4. **事件** — 好感度 ≥ 60 触发拜师、结婚、结拜
   - **拜师**后向师傅最高属性学习，获得 5–10 + 师傅属性/20 点该属性
5. **奇遇** — 宝物按类别独占，同类新宝物替换旧宝物
6. **侠义值** — 完成任务增加 2~6
7. **不进则退**（每 10 回合）— 选择一项属性降低 10%
8. **天下第一武道会**（每 30 回合）— 侠义值 Top 16 参赛
9. **战斗系统** — 战术对抗 + 地形影响 + 多回合推演
10. **势力扩张** — AI/玩家均可攻占城池，扩张势力
11. **经济系统** — 城池发展、人口增长、驻军、兵力招募
12. **江湖移动**（每 2 回合）— 角色按城市路线移动
13. **江山舆图**（增强）— Canvas 交互地图 + 城池控制可视化
14. **称号系统** — 13 种自动称号
15. **结算画面** — 展示称号、属性变化、好友宿敌、宝物、江湖历程
16. **托管模式** — 自动执行每回合
17. **玩家阵亡** — 攻城战中死亡触发游戏结束结算画面
18. **AI 防御同盟** — 城主人之间好感度 > 30 的 AI 自动结盟，盟友被攻击时派遣 40% 兵力驰援

## 战斗系统

### 战术体系

6 种战术，各有攻防倍率和士气消耗：

| 战术 | 攻击倍率 | 防御倍率 | 士气消耗 | 描述 |
|------|---------|---------|---------|------|
| 正面强攻 | 1.3 | 0.7 | 10 | 全军突击，不计损失 |
| 稳步推进 | 1.0 | 1.0 | 5 | 稳扎稳打，步步为营 |
| 迂回包抄 | 1.2 | 0.8 | 8 | 分兵绕后，夹击敌军 |
| 据险防守 | 0.7 | 1.4 | 3 | 依托地形，固守待援 |
| 诱敌深入 | 1.5 | 0.5 | 15 | 佯败诱敌，伏兵四起 |
| 火攻 | 1.4 | 0.6 | 12 | 借助风势，火攻敌军 |

### 战术克制（攻击方视角）

正面强攻→据险防守 0.6 / 迂回包抄 1.3 · 稳步推进→正面强攻 0.8 / 诱敌深入 1.2 · 迂回包抄→据险防守 1.4 / 正面强攻 1.1 · 据险防守→正面强攻 1.2 / 火攻 0.5 · 诱敌深入→正面强攻 1.5 / 稳步推进 0.7 · 火攻→据险防守 1.5 / 迂回包抄 0.8

### 战斗流程

1. 计算双方战力（兵力 × 训练 × 士气 + 武力 + 地形加成）
2. 防守方获得城池驻军加成
3. AI 根据战力比、地形、性格选择战术
4. 战术对抗计算最终攻击倍率
5. 最多 5 回合推演，每回合双方按战力比损失兵力
6. 胜负判定：一方兵力归零则败；超 5 回合按损失比例判定
7. 战死判定：兵力归零且士气 < 10（40% 阵亡），或损失 > 70% 且士气 < 5（25% 阵亡）
8. 胜利方吸收部分守军并占领城池

## 经济与城战系统

### 城池属性

18 座城池各有：**开发度** (0-100)、**人口**、**驻军**、**地形**（平原/山地/水域/关隘/沙漠）、**控制者**

### 经济循环

- **每回合**：开发度 +0.01~0.03、人口 +0.2%~0.7%
- **每 5 回合**：驻军 +5%、武将征兵（控制城池 × 人口 × 开发度系数）
- **无城池武将**：每 5 回合自动获得 10~30 散兵

### 招募士兵

- **玩家**：每回合 35% 概率出现「招募士兵」任务，成功 +500 兵力，受魅力影响
- **NPC**：每回合 30% 概率自动尝试招募，成功 +500 兵力，受魅力影响

### AI 扩张（每 3 回合，兵力需 > 2000）

1. 从所在地出发，查找相邻"无主"或"敌对"城池
2. 不攻击盟友城池
3. 优先攻打兵力弱于己方 1.2 倍的目标
4. 使用战斗引擎模拟→胜利则占领
5. 每场战斗公告 + 战报日志

### AI 防御同盟（每 5 回合）

1. 仅城主人之间结盟，好感度 > 30 的 AI 优先配对
2. 好感度越高的组合优先结盟
3. 盟友被攻击时自动派遣 40% 兵力驰援
4. AI 不会攻击盟友的城池

### 玩家扩张

1. 兵力 > 2000 时，40% 概率「攻城略地」任务出现在任务列表
2. 目标为所在地相邻的城池（无主或敌对），遇守军使用完整战斗引擎
3. 胜利后自动占领目标城池，吸收部分驻军
4. 防守方若有盟友，盟友自动派遣 40% 兵力驰援
5. 玩家在攻城战中阵亡 → 游戏结束

## 城市系统

18 座城市分布于五大区域，角色出生时分配至所属区域内的城市（仅作为所在地，无控制权）：

| 区域 | 城市 |
|------|------|
| 中原 | 洛阳、开封、长安、燕京、应天府**(帝都)**、襄阳 |
| 江南 | 苏州**(中心→襄阳)**、杭州、扬州 |
| 塞北 | 凉州**(中心→燕京)**、朔方、云中 |
| 西域 | 敦煌**(中心→长安)**、龟兹、于阗 |
| 东海 | 登州**(中心→应天府)**、泉州、宁波 |

> 初始仅皇帝朱棣占据应天府（5000 兵力），其余城池无主。所有角色初始兵力 500。通过招募士兵（+500/次）积累兵力 > 2000 后，可概率触发攻城略地任务，从所在地攻打相邻城池。

中原四座关隘城市（长安、燕京、应天府、襄阳）连成环形。区域中心通过关隘连接中原。外环连接：
- 燕京↔云中、凉州↔燕京（中原↔塞北）
- 应天府↔扬州、长安↔于阗、长安↔凉州、敦煌↔凉州（中原↔西域/江南/塞北）
- 开封↔燕京、登州↔燕京、登州↔扬州、扬州↔杭州、扬州↔襄阳（多条外环辐射）
- 苏州↔泉州、苏州↔于阗（江南↔东海/西域）

## 江山舆图（交互地图）

点击导航栏「江山舆图」打开：

- **Canvas 背景**：加载 `map.png` 为底图，半透明遮罩保证可读性
- **城池标注**：按坐标绘制圆形节点（中原帝都更大圆），白色名称标签
- **连线**：城池之间淡蓝色连线表示路线
- **控制者颜色**：绿色（玩家）、红色（敌方 NPC）、灰色（无主）
- **控制者标签**：金色字体显示控制者名称
- **文本阴影**：深色描边确保地图上文字清晰
- 下方列表同步显示开发度、驻军、控制者

## 战报系统

点击事件栏中的 ⚔ 战斗/征服事件，弹出战报弹窗：

- **胜负标头**：攻击方胜利（绿色）/ 防守方胜利（红色）/ 平局（橙色）
- **战术摘要**：双方选用战术、战术倍率、兵力损失
- **回合详情**：5 回合逐轮展示损失与剩余兵力
- **战报日志**：完整文字记录从战术选择到胜负判定

## 属性系统

- **机敏(l)** — 非人物类任务成功率
- **武力(w)** — 战斗力，≥150 获「武道宗师」
- **魅力(i)** — 人物类任务成功率
- **智谋(p)** — 谋略任务加成（系数 × 0.1）
- **侠义值** — 通过任务积累，≥50→侠义之士，≥100→一代大侠
- 角色卡片新增：**兵力**、**士气**、**训练度**、**控制城池** 徽章

## 好感度系统

- 初始好感随机范围 -20~20
- 合作任务成功 +20~40，失败 -10~20
- 拜师/结婚/结拜需好感 ≥ 60

## 重置与重新开始

- **游戏结束** → 点击「重新开始」→ 调用 `/api/new_game` 完全重置世界状态 + 引擎状态 → 弹出选人界面
- 新游戏为全新世界（所有角色复活、城池重置、回合归零）
- 游戏结束触发条件：玩家在攻城战中阵亡、极难任务失败死亡、华山论剑结束

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/world` | 获取世界状态（回合、角色、玩家、好感度、城池经济） |
| GET | `/api/start/candidates` | 获取初始选人候选（3人） |
| POST | `/api/new_game` | 重置世界 + 引擎（重新开始） |
| POST | `/api/start/select` | 选择角色 `{"name":"黄宇翔"}` |
| POST | `/tick` | 推进一回合 |
| GET | `/api/events/pending` | 获取待处理大事 |
| POST | `/api/events/decide` | 处理事件 |
| GET | `/api/events` | 获取江湖日志（含战报事件） |
| GET | `/api/tasks/pending` | 获取本回合任务列表（含扩张任务） |
| POST | `/api/tasks/execute` | 执行任务 `{"task_id":0,"mode":"normal\|scheme\|focus"}` |
| GET | `/api/decay/pending` | 获取不进则退待选属性 |
| POST | `/api/decay/execute` | 选择降低属性 |
| GET | `/api/tournament` | 获取武道会对阵图及结果 |
| GET | `/api/move/pending` | 获取待选移动路线 |
| POST | `/api/move` | 执行移动 `{"dest":"洛阳"}` |
| GET | `/api/data/map.png` | 江山舆图地图图片 |
