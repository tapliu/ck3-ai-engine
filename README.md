# 浪子江湖 · 侠行四海 v1.2.0

AI 驱动的武侠江湖模拟&策略游戏。35 位角色（+ 自定义武将）出自小说《浪子江湖》，具备回合制战斗、经济、城池经营、帝国更迭、扩张等系统。

## 快速开始

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

浏览器打开 `http://localhost:8000`

> 图片自动挂载到 `/api/images/`，支持 WebP 格式快速加载。

## 游戏玩法

1. **选人** — 开局从 3 个随机角色 + 1 个「自建武将」选项中选择你的侠客；每张角色卡展示姓名、性别、年龄、四项属性、评分、头像、以及**搭配的武学技能名称与描述**
2. **Tick** — 每点击一次推进一回合，NPC 自主决策（战争/刺杀/结盟等）
3. **任务** — 每回合 1–3 个江湖任务，分七类：
   - **非人物类** — 成功主属性：机敏
   - **人物类** — 成功主属性：魅力
   - **合作任务** — 同城 50%/同区域 25% 概率将普通任务转为合作
   - **极难任务** (30% 概率) — 高风险高回报，失败可能死亡
   - **招募士兵**（概率触发）— 消耗金钱，成功获得兵力受城池人口和金钱投入影响
   - **攻城略地**（兵力 > 2000 时概率出现）— 从所在地攻打相邻城池（无主或敌对），进入回合制战斗
     - **无主城**触发城防守军战（Garrison 对象），不再是秒占
   - 三种执行模式：普通 / 用谋略（智谋加成，3 回合冷却）/ 全力施展（3 回合冷却）
     - 谋略/全力模式无论成败均进入冷却（包括招募和攻城任务）
4. **事件** — 好感度 ≥ 60 触发拜师、结婚、结拜
   - **拜师**后向师傅最高属性学习，获得 5–10 点该属性（师傅属性越高增益越大）
5. **奇遇** — 宝物按类别独占，同类新宝物替换旧宝物，单属性加成 ≤10，总加成 ≤20
6. **侠义值** — 完成任务增加 2~6
7. **不进则退**（每 10 回合）— 选择一项属性降低 10%
8. **天下第一武道会**（每 30 回合）— 侠义值 Top 16 参赛
9. **战斗系统** — 战术对抗 + 地形影响 + 多回合推演 + **玩家回合制战斗**
10. **势力扩张** — AI/玩家均可攻占城池，扩张势力；**帝都（应天府）极难攻陷**（防守方 2×属性），攻陷帝都触发**改朝换代**，新帝全属性 +10
11. **经济系统** — 城池发展、人口增长、驻军、兵力招募、**税金**（每回合武将付 2% 金给城主）
12. **江湖移动**（每 2 回合）— 角色按城市路线移动；移动面板显示**城主名**
13. **江山舆图**（增强）— Canvas 交互地图 + 城池控制可视化 + 阵营颜色标识
14. **称号系统** — 13 种自动称号，部分称号奖励金和侠义值
15. **结算画面** — 展示称号、属性变化、好友宿敌、宝物、江湖历程
16. **托管模式** — 自动执行每回合
17. **玩家阵亡** — 攻城战中死亡触发游戏结束结算画面
18. **AI 防御同盟** — 城主人之间好感度 > 30 的 AI 自动结盟，盟友被攻击时派遣 40% 兵力驰援
19. **自建武将** — 在主菜单「编辑武将」中创建/编辑自定义角色，自由分配属性（1–200），从 10 套头像/全身像中选择形象；开局选人界面可直接选择自建武将

## 战斗系统

### 武将定位

根据最高属性（机敏/武力/智谋）自动获得称号：
- **鬼谷子** — 机敏最高（防御型）
- **万人敌** — 武力最高（攻击型）
- **智多星** — 智谋最高（技能型）

### 技能系统

每位武将随机获得一个武学技能，技能池共 **17 种**：

**伤害类（10种）**：烈焰斩、穿心箭、雷霆击、剑气纵横、破天锤、寒冰掌、断浪刀、碎星拳、追风剑 — 伤害公式 `0.25 × 攻击方智谋(p)/防守方智谋(p)`，造成百分比兵力伤害
- **沉默类（4种）**：封喉诀、哑穴点、金针封脉、闭气诀 — 使对手 2 回合无法使用技能
- **禁锢类（3种）**：定身术、迷魂咒、天罗网、点穴手 — 使对手 1 回合无法行动

所有技能使用后进入 **2 回合冷却**，冷却中技能按钮不可用，冷却回合数在战斗界面显示。技能命中率 = `智谋(p)/200`。

### 属性与战斗

- **武力(w)** → 攻击力：决定普攻伤害
- **机敏(l)** → 防御力：减少受到的伤害
- **智谋(p)** → 技能威力/命中率

### 玩家回合制战斗

玩家触发「攻城略地」或防御已占领城池时，进入回合制战斗界面：

1. **战前选阵** — 从 6 种战术中选择，战术影响本次战斗的攻防倍率
2. **回合行动** — 每回合可选：攻击/防御/技能/撤退
3. **实时显示** — 双方兵力（HP血条）、攻击力、防御力、状态效果、技能冷却回合
4. **战术克制** — 被克制一方攻击防御小幅下降（约 15-30%）
5. **胜负结算** — 胜利/败北/平局/撤退，影响士气变化
6. **驻军选择** — 攻占城池后选择留守兵力（从当前兵力扣除）

### 防守方减伤

当对手选择防御动作时，攻击方伤害减半（对玩家和 AI 均生效）。

### AI 撤退

AI 兵力低于 500 时，40% 概率选择撤退。撤退后战斗结束，攻击方获胜（不占领城池，无驻军选择）。

### 战后恢复

战斗结束后，双方均有机会恢复部分损失的兵力：`恢复比例 = clamp(0.1 + gold/5000, 0.1, 0.3)`，消耗 `恢复兵力 × 1.5` 金。

### 帝都攻防

当防守方是皇帝且防守城市为帝都（应天府）时，防守方攻防属性 ×2。AI 需要 5 倍兵力才会尝试攻打帝都。

### 战后士气

| 结果 | 攻击方 | 防守方 |
|------|--------|--------|
| 胜利 | +15(受魅力加成) | -15 |
| 失败 | -15(受魅力影响) | +15 |
| 撤退 | -15(受魅力影响) | +10 |
| 平局 | -5 | -5 |

### 士气浮动（每回合）

- 士气 < 30：向 30 恢复，`5 + 魅力×0.05`
- 士气 > 30：向 30 衰减，`10 - 魅力×0.05`

### 战术体系（AI 通用）

6 种战术，各有攻防倍率和士气消耗：

| 战术 | 攻击倍率 | 防御倍率 | 士气消耗 | 描述 |
|------|---------|---------|---------|------|
| 正面强攻 | 1.3 | 0.7 | 10 | 全军突击，不计损失 |
| 稳步推进 | 1.0 | 1.0 | 5 | 稳扎稳打，步步为营 |
| 迂回包抄 | 1.2 | 0.8 | 8 | 分兵绕后，夹击敌军 |
| 据险防守 | 0.7 | 1.4 | 3 | 依托地形，固守待援 |
| 诱敌深入 | 1.5 | 0.5 | 15 | 佯败诱敌，伏兵四起 |
| 火攻 | 1.4 | 0.6 | 12 | 借助风势，火攻敌军 |

### 战术克制（攻击方视角）

正面强攻→据险防守 0.6 / 迂回包抄 1.3 · 稳步推进→正面强攻 0.8 / 诱敌深入 1.2 · 迂回包抄→据险防守 1.4 / 正面强攻 1.1 · 据险防守→正面强攻 1.2 / 火攻 0.5 · 诱敌深入→正面强攻 1.5 / 稳步推进 0.7 · 火攻→据险防守 1.5 / 迂回包抄 0.8

### 战斗流程

1. 计算双方战力（兵力 × 训练 × 士气 + 武力 + 地形加成）
2. 防守方获得城池驻军加成
3. AI 根据战力比、地形、性格选择战术
4. 战术对抗计算最终攻击倍率
5. 最多 5 回合推演，每回合双方按战力比损失兵力
6. 胜负判定：一方兵力归零则败；超 5 回合按损失比例判定
7. 战死判定：兵力归零且士气 < 10（40% 阵亡），或损失 > 70% 且士气 < 5（25% 阵亡）
8. 胜利方吸收部分守军并占领城池

## 经济与城战系统

### 城池属性

18 座城池各有：**开发度** (0-100)、**人口**、**驻军**、**地形**（平原/山地/水域/关隘/沙漠）、**控制者**

### 经济循环

- **每回合**：开发度 +0.01~0.03、人口 +0.2%~0.7%、城池金收入（开发度×3 + 人口/1000）
- **每 5 回合**：驻军 +5%、武将征兵（控制城池 × 人口 × 开发度系数）
- **无城池武将**：每 5 回合自动获得 10~30 散兵

### 税金系统

每回合，每位活着的武将支付 **2% 当前金钱（最少 1 金）** 给其所在城市的控制者（城主），城主必须存活；若城市无主或付款人为城主本人，则不产生税金。

### 金钱系统

- 每位武将初始 200 金
- 城池每回合产生金收入（开发度×3 + 人口/1000）
- 招募士兵消耗金钱（50 + 难度×5）
- 普通任务奖励 5-20 金，极难任务 20-50 金
- 称号奖励可能包含金钱和侠义值

### 招募士兵

- **玩家**：每回合 35% 概率出现「招募士兵」任务，消耗金钱，获得兵力 `gain = cost × population / 6000`，上限 20-1000
- **NPC**：每回合 30% 概率自动尝试招募（使用相同减半公式）

### AI 扩张（每 3 回合，兵力需 > 2000）

1. 从所在地出发，查找相邻"无主"或"敌对"城池
2. 不攻击盟友城池
3. 优先攻打兵力弱于己方 1.2 倍的目标
4. 使用战斗引擎模拟→胜利则占领
5. 每场战斗公告 + 战报日志

### AI 防御同盟（每 5 回合）

1. 仅城主人之间结盟，好感度 > 30 的 AI 优先配对
2. 好感度越高的组合优先结盟
3. 盟友被攻击时自动派遣 40% 兵力驰援
4. AI 不会攻击盟友的城池

### 玩家扩张

1. 兵力 > 2000 时，40% 概率「攻城略地」任务出现在任务列表
2. 目标为所在地相邻的城池（无主或敌对），遇守军进入**回合制战斗**
   - **无主城**不再秒占，改为与城防守军（Garrison，属性由城池开发度决定）交战
3. 战前选择战术 → 战术克制影响攻防 → 每回合选择攻击/防御/技能/撤退
4. 胜利后弹出**驻军选择**界面，拖动滑块决定留守兵力
5. 防守方若有盟友，盟友自动派遣 40% 兵力驰援
6. 玩家在攻城战中阵亡 → 游戏结束

### 帝都攻陷与改朝换代

攻克帝都（应天府）触发**改朝换代**：
- 旧皇帝失去 `is_emperor` 身份
- 征服者成为新皇帝，`is_emperor = True`
- 新皇帝全部四项基础属性 +10
- 系统广播「改朝换代」事件

## 城市系统

18 座城市分布于五大区域，角色出生时分配至所属区域内的城市（仅作为所在地，无控制权）：

| 区域 | 城市 |
|------|------|
| 中原 | 洛阳、开封、长安、燕京、应天府**(帝都)**、襄阳 |
| 江南 | 苏州**(中心→襄阳)**、杭州、扬州 |
| 塞北 | 凉州**(中心→燕京)**、朔方、云中 |
| 西域 | 敦煌**(中心→长安)**、龟兹、于阗 |
| 东海 | 登州**(中心→应天府)**、泉州、宁波 |

> 初始仅皇帝朱棣占据应天府（5000 兵力），其余城池无主。所有角色初始兵力 500。通过招募士兵（+500/次）积累兵力 > 2000 后，可概率触发攻城略地任务，从所在地攻打相邻城池。

中原四座关隘城市（长安、燕京、应天府、襄阳）连成环形。区域中心通过关隘连接中原。外环连接：
- 燕京↔云中、凉州↔燕京（中原↔塞北）
- 应天府↔扬州、长安↔于阗、长安↔凉州、敦煌↔凉州（中原↔西域/江南/塞北）
- 开封↔燕京、登州↔燕京、登州↔扬州、扬州↔杭州、扬州↔襄阳（多条外环辐射）
- 苏州↔泉州、苏州↔于阗（江南↔东海/西域）

## 江山舆图（交互地图）

点击导航栏「江山舆图」打开：

- **Canvas 背景**：加载 `map.png` 为底图，半透明遮罩保证可读性
- **城池标注**：按坐标绘制圆形节点（中原帝都更大圆），白色名称标签
- **连线**：城池之间淡蓝色连线表示路线
- **控制者颜色**：绿色（玩家）、红色（敌方 NPC）、灰色（无主）
- **控制者标签**：金色字体显示控制者名称
- **文本阴影**：深色描边确保地图上文字清晰
- 下方列表同步显示开发度、驻军、控制者

## 战报系统

点击事件栏中的 ⚔ 战斗/征服事件，弹出战报弹窗：

- **胜负标头**：攻击方胜利（绿色）/ 防守方胜利（红色）/ 平局（橙色）
- **战术摘要**：双方选用战术、战术倍率、兵力损失
- **回合详情**：5 回合逐轮展示损失与剩余兵力
- **战报日志**：完整文字记录从战术选择到胜负判定

## 属性系统

- **机敏(l)** — 防御力、非人物类任务成功率
- **武力(w)** — 攻击力、战斗力，≥150 获「武道宗师」
- **智谋(p)** — 技能威力和命中率、谋略任务加成
- **魅力(i)** — 人物类任务成功率、士气浮动速率
- **侠义值** — 通过任务积累，≥50→侠义之士，≥100→一代大侠
- **金钱(gold)** — 招募士兵消耗、任务奖励、城池收入
- 角色卡片新增：**兵力**、**士气**、**训练度**、**金钱**、**控制城池**、**主属性称号**、**技能** 徽章

## 好感度系统

- 初始好感随机范围 -20~20
- 合作任务成功 +20~40，失败 -10~20
- 拜师/结婚/结拜需好感 ≥ 60

## 重置与重新开始

- **游戏结束** → 点击「重新开始」→ 调用 `/api/new_game` 完全重置世界状态 + 引擎状态 → 弹出选人界面
- 新游戏为全新世界（所有角色复活、城池重置、回合归零）
- 游戏结束触发条件：玩家在攻城战中阵亡、极难任务失败死亡、华山论剑结束

| GET | `/api/world` | 获取世界状态（回合、角色、玩家、好感度、城池经济、`active_battle`） |
| GET | `/api/start/candidates` | 获取初始选人候选（3人，每人带 skill 字段） |
| POST | `/api/new_game` | 重置世界 + 引擎（重新开始） |
| POST | `/api/start/select` | 选择角色 `{"name":"黄宇翔"}` |
| POST | `/tick` | 推进一回合 |
| GET | `/api/events/pending` | 获取待处理大事 |
| POST | `/api/events/decide` | 处理事件 |
| GET | `/api/events` | 获取江湖日志（含战报事件） |
| GET | `/api/tasks/pending` | 获取本回合任务列表（含扩张任务） |
| POST | `/api/tasks/execute` | 执行任务 `{"task_id":0,"mode":"normal\|scheme\|focus"}` |
| GET | `/api/decay/pending` | 获取不进则退待选属性 |
| POST | `/api/decay/execute` | 选择降低属性 |
| GET | `/api/tournament` | 获取武道会对阵图及结果 |
| GET | `/api/move/pending` | 获取待选移动路线 |
| POST | `/api/move` | 执行移动 `{"dest":"洛阳"}` |
| GET | `/api/data/map.webp` | 江山舆图地图图片（WebP） |
| GET | `/api/characters/edit` | 获取所有武将当前数值（编辑模式） |
| POST | `/api/characters/save` | 保存武将编辑 `[{id,name,gender,l,w,i,p,age,desc}]` |
| POST | `/api/characters/reset` | 重置所有武将为初始值 |
| GET | `/api/characters/custom` | 获取自建武将列表 |
| POST | `/api/characters/custom/create` | 创建自建武将 |
| POST | `/api/characters/custom/update` | 更新自建武将 |
| POST | `/api/characters/custom/delete` | 删除自建武将 |
| POST | `/api/battle/start_tactic` | 回合制战斗选阵 `{"tactic":"正面强攻"}` |
| POST | `/api/battle/action` | 回合行动 `{"action":"attack\|defend\|skill\|retreat"}` |
| POST | `/api/battle/garrison` | 攻占城池后驻军 `{"garrison":500}` |
| GET | `/api/battle/state` | 查询战斗状态 |
| POST | `/api/save` | 存档（pickle 序列化） |
| POST | `/api/load` | 读档 |

## 环境变量

| 变量 | 说明 |
|------|------|
| `OPENAI_API_KEY` | OpenAI API 密钥（不设置时 NPC 默认 wait，游戏可正常运行） |

## 数据来源

角色数据来源于舒志琪小说《浪子江湖》，包含 35 位角色及 20 件宝物。
